
## 🚀 Quick Start (Phase 1 — Frontend)

### Prerequisites
- Node.js ≥ 18  ([download](https://nodejs.org))
- Git

### Run locally

```bash
# 1. Clone
git clone https://github.com/YOUR_USERNAME/codewithiit.git
cd codewithiit/frontend

# 2. Install dependencies
npm install

# 3. Start dev server
npm run dev

# 4. Open browser at:
#    http://localhost:5173
```

That's it. No backend needed for Phase 1 — progress saves to localStorage.
